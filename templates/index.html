Many To Many Assignment problem, by improving the Khun-Munkers algorithm with Backtracking.




Description of the Algorithm

The algorithm is an improvement of the Khun-Munkers algorithm, which is used to solve the assignment problem. The assignment problem is a combinatorial optimization problem that consists of finding the best assignment of agents to tasks. Our improvement consists of using backtracking to find the optimal solution, while alloing each agent to be assigned to multiple tasks, and each task to be assigned to multiple agents.
